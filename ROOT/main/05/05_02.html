사이버추모관 안내

  • 사이버추모관은 시간과 장소의 제약을 받지 않고, 고인의 영혼을 추모하고자하는 모든 분들을 위해 마련된 인터넷상의 가상공간을 말합니다.

하늘편지

  • 누구나 이용가능하며, 작성하신 추모글을 공개/비공개 설정하실 수 있습니다.

추모의방

  • 익산추모관에 안치된 고인의 유가족에 한하여 개설이 가능하며, 가족 외의 분들께서는 개설자에게 별도 이용자등록 요청 후 이용이 가능합니다.

신청절차
로그인 로그인 버튼을 클릭하여 본인인증을 완료합니다.
신청서작성 아래의 신청서 작성 버튼을 클릭하여 추모의방 개설에 필요한 정보 기재 후 제출버튼을 눌러 완료합니다.
승인 제출 신청서를 익산추모관 담당자가 검토 후 승인이 진행됩니다.
이용자등록 추가 이용자 등록은 추모의방 개설자만 가능하며, 추모의방 이용자추가 버튼을 통해 등록합니다.

    추모 외의 목적으로 이용 시 이용권한이 취소됨은 물론 폐쇄(삭제) 조치 됩니다.

  • 신청자께서는 반드시 항상 연락 가능한 E-mail 주소를 갖고 계셔야 합니다
